Commande Bash
==============

ls => Affiche le contenue du dossier / ls -lah 
cd => Permet de naviguer dans les dossiers / cd .. => Précédent dossiers / ex : cd Document
mkdir => Crée un dossier / ex : mdkir nouveau_dossier
rm => Supprime le dossier
clear => Clear le terminal 
git init => Crée un dépôt Git
touch => Crée un fichier/ ex : touche tutoriel.txt
git status => Status actuel du dépot
git commit => Sauvegarde des fichier sélectionner "Nom de la sauvegarde" -m"..."
